# Inventory and User Management System (Storage System)

## About the Project
This project is a collaborative effort developed to solve real-world data management challenges. We built a comprehensive management system that allows Administrators and Employees to efficiently manage user data and inventory records using C++.

## Key Features
- Login system with role-based access control (Admin & Employee).
- Full user management (Add, Search, Edit, Delete).
- Inventory management (Record items, Search, Sort records, Update price and quantity).
- Data Persistence: Save and load data from text files.
- Advanced C++ Concepts: Pointers, Structs, Namespaces, Static Variables, and File Handling.

## Technologies Used
- Language: C++
- Tools: File Handling, Algorithm Design (Bubble Sort)

## How to Run?
1. Ensure you have a C++ compiler installed (e.g., Dev-C++, VS Code).
2. Download the `main.cpp` file.
3. Compile and run the program to explore the system!
